    创建个站步骤

    创建个人网站基本步骤

    1.安装mkdocs,实现本地部署 mkdocs部署基本流程
    2.创建Github仓库及安装Git, 通过Git部署本地内容至Github仓库
    3.更改主题，可参考Material for MkDocs

    文件部署到仓库指令

    git init  
git add .  
git commit -m "init repo"

    这里的 init repo 可以更换成任意描述，用来概括你此次提交代码的行为。

    git branch -M main

    git remote add origin https://github.com/WangTao19950912/WangTao19950912.github.io

    不是每次必须要建立连接

    git push -u origin main

    如果强制推送命令

    git push -u origin main --force

    将本地照片等资源上传到CDN

    CDN (Content Delivery Network) ，此处使用jsDelivr

    基本步骤

    1.创建仓库
    2.安装PicGo PicGo下载链接
    3.配置PicGo PicGo+Github配置及使用文档
    4.上传图片

    遇到的问题

    1.本地文件更改后，直接通过git push -u origin main一直不能上传到仓库， 是因为不了解Git的机制，本地文件夹不是直接上传至github，而是通过Git上传，需要先将本地文件git执行才可以上传

    2. PicGo不能上传图片，检查日志，询问deepseek解决
    3. git config
    4. 切换git登录方式为通过SSH密钥登录
